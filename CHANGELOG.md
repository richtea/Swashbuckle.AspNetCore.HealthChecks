# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](http://keepachangelog.com/)
and this project adheres to [Semantic Versioning](http://semver.org/).

## [1.0.0] - 2025-12-14

### Added

- Support for .NET 10 now available.

### Removed

- No longer supported for .NET versions prior to .NET 8.0. To use this library with unsupported .NET versions, choose an
  earlier version of the package.

## [0.1.0] - 2022-12-18

### Changed

- Use license expression instead of embedding LICENSE file, to work better with NuGet Gallery.

## [0.1.0-beta.0] - 2022-12-12

### Added

- Add multi-targeting to support the same .NET versions as Swashbuckle itself, except .NET 5 which is no longer in
  support.

## [0.1.0-alpha.0] - 2022-12-11

### Added

- Initial version
